# Project: PromptLab Preset Showcase

## 背景和动机
用户需要一个网页来展示Prompt预设，包含提示词（中英双语）、图片和分类。
核心需求包括：
- 展示预设（图片、提示词、分类）。
- 检索和分类筛选。
- 复制提示词功能。
- 中英双语切换。
- **管理后台**：通过界面添加/编辑预设，而非代码操作。
- UI风格参考 `styles/podui.css`。

## 关键挑战和分析
1.  **数据持久化**：用户希望通过界面编辑，这意味着前端需要一种方式写入数据。由于是本地“Lab”环境，最简单的方案是使用一个轻量级后端（Node.js/Express）来读写本地 JSON 文件 (`presets.json`)。
2.  **UI 风格整合**：需深度集成 `podui.css`，确保组件（卡片、按钮、输入框）符合设计规范（深色玻璃拟态风格）。
3.  **双语切换**：状态管理需支持全局语言切换，并在组件层面响应。
4.  **图片管理**：上传图片需要处理。简单方案是将图片保存到 `public/images` 或 `uploads` 目录。后端需支持文件上传。
